# Onboarding Notes

This upstream text explains the default invitation and profile model for deployments that want longer onboarding copy outside JSON config.

New users should understand:

- the app is a community sharing and borrowing portal;
- access can require an invite or operator approval;
- a profile helps admins and borrowers identify responsible participants;
- item contributions may be moderated before or after publication;
- deployment-specific operators must replace this text before production use.
