# Legacy Agent Directory

This directory contains older agent skills that may still be useful as reference material for some IDEs.

The current repository source of truth is `.agents/`:

- Rules: `.agents/rules/`
- Workflows: `.agents/workflows/`
- Skills: `.agents/skills/`

Before using or extending any file in this legacy `.agent/` directory, compare it with `.agents/` and migrate useful guidance there. Do not treat CONTEKT-era or tool-specific assumptions in legacy skills as upstream policy.
